Fetch a dataset

This webapp probably needs to get a dataset from Dataiku DSS. You can do that using the Javascript API.
If you want to use more advanced features, you can even use the REST API directly.

To interact with Dataiku, the webapp must identify itself. To do that, we created an API key.
For security reasons, this API key is not allowed to do much by default. Before you can fetch a dataset, you must explicitly allow it in the settings tab of the webapp editor.
